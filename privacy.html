Privacy Policy

Effective date: 2026-05-03

What the extension processes

The extension reads Genspark conversation content from pages you open in your browser when you initiate an export. This can include chat messages, prompts, AI responses, links, images, and file references shown on the Genspark page.

What stays local

Conversation content is exported to local markdown/ZIP files on your device. Chat content is not uploaded to our servers.

Network requests

  • Optional asset downloads from HTTPS *.genspark.ai URLs referenced in the conversation (Pro only).
  • License key validation requests to the build-configured validation endpoint (Pro licensing).

Data stored in extension storage

  • license key and cached license status
  • license validation timestamps
  • daily free-tier export counters
  • checkout URL settings

Chrome Web Store data categories

  • Authentication information: Pro license key only.
  • Personal communications: Genspark chat messages processed for local export.
  • Website content: text, links, images, files, and other page content from the Genspark conversation being exported.

Data sharing and sale

We do not sell conversation data or share exported chat content with third parties. Pro license validation sends the entered license key to the validation endpoint; it does not send chat content.

Contact

For privacy concerns, open an issue on our GitHub support page.
